# 家庭画像 Agent 作答试卷

本文是一份用于评测画像算法下游效果的试卷。评测对象不是画像算法的内部格式，而是 agent 在接入某个画像算法或记忆查询接口后，能否回答关于家庭画像和居民画像的问题。

本试卷不向 agent 提供标准答案。agent 只能使用被测算法提供的查询接口、画像报告或 memory 工具作答。

## 使用方式

评测时保持以下条件一致：

1. 所有算法使用同一份事件数据或同一批 memory 输入。
2. 所有算法面向 agent 暴露各自查询接口，但 agent 的系统提示词、模型、查询次数预算保持一致。
3. agent 不得读取生成器 truth、persona、scenario 或标准答案文件。
4. agent 可以回答“不确定”“无法判断”“需要更多证据”。
5. 评分看最终答案质量，不要求算法输出证据链、置信度或统一结构。

建议 agent 每题回答：

- 结论。
- 简短理由。
- 如果不能确定，说明不能确定的边界。

## 评分说明

每题 3 分，总分 60 分。

- 3 分：关键结论完整，边界合理，没有明显越界。
- 2 分：主要方向正确，但遗漏一个关键点或表达略过度。
- 1 分：只答到少量相关信息，或混入明显不确定/错误内容。
- 0 分：核心方向错误、答非所问，或输出严重越界结论。

## A. 家庭结构与成员槽位

### Q1. 这个家庭整体上更像什么样的家庭组成？

请描述家庭规模倾向、可能的成员槽位，以及是否存在宠物候选。

### Q2. 是否可以判断家中存在老人？

如果可以，请说明结论；如果不可以，请说明为什么不应这样判断。

### Q3. 这个家庭中可能有哪些主要居民槽位？每个槽位大致承担什么生活角色？

请不要使用真实姓名；可以使用“通勤成年人”“白天在家办公槽位”“儿童活动槽位”等描述。

### Q4. 是否可以把所有家庭活动归属于同一个人？

请说明是否应拆分多个居民槽位，以及原因。

## B. 工作日与周末生活节律

### Q5. 工作日早晨通常发生什么流程？

请覆盖起床、早餐、上学或通勤离家等关键阶段。

### Q6. 工作日白天家中是否有稳定活动？主要发生在哪里？

请说明可能的活动类型和相关空间。

### Q7. 工作日晚间的主要 routine 是什么？

请覆盖晚餐准备、家庭共处、儿童睡前和成年人睡眠等节奏。

### Q8. 周末节奏与工作日有什么不同？

请说明起床/早餐、外出或居家娱乐、睡觉时间等差异。

## C. 空间使用画像

### Q9. 书房在这个家庭中最可能承担什么功能？

请说明它更像普通偶发活动空间，还是稳定功能空间。

### Q10. 厨房在不同时间段承担哪些功能？

请区分工作日早餐、工作日晚餐、周末早午餐等场景。

### Q11. 客厅、儿童房、主卧分别承担什么主要功能？

请分别回答，不要把房间名称直接当作唯一证据。

### Q12. 花园相关活动应该如何解释？

请说明它可能关联哪些家庭行为，以及哪些解释需要谨慎。

## D. 设备、家务与使用习惯

### Q13. 厨房设备之间是否存在稳定联动？

请重点回答冰箱、灶台、油烟机在做饭场景中的关系。

### Q14. 扫地机器人和洗衣相关习惯有什么特点？

请说明它们是否每天发生、通常在什么上下文下发生。

### Q15. 门锁事件可以支持哪些家庭结论？又不能单独推出什么？

请回答离家/回家、安全习惯，以及门锁事件的推断边界。

### Q16. 环境传感器和空调类信号应如何使用？

请说明哪些可以作为家庭偏好或环境上下文，哪些不应直接当成人为行为。

## E. 安全、自动化与服务机会

### Q17. 如果只推荐一个高价值自动化场景，你会推荐什么？

请说明触发条件、动作和风险提醒。

### Q18. 哪些情况应该被视为安全风险或异常提醒？

请覆盖厨房、门锁、水流、夜间异常或类似风险。

### Q19. 宠物或花园摄像头活动是否可以直接判断为安全入侵？

请说明合理解释和不应过度推断的部分。

## F. 边界与不可判断题

### Q20. 请列出至少 5 个当前画像不应高确定性输出的结论。

这些结论可以是身份、角色、人数、健康、安全或长期趋势相关内容。
